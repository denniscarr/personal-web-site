Professional Work

Rush Hour

An adaptation of the classic board game by ThinkFun into a daily web format, made for TED Conferences.

I programmed this game in Phaser with a React/TypeScript wrapper, and handled page design and UI/UX in collaboration with the team at TED .

Spellcheck

A daily spelling web game made for TED Conferences based on a concept by Answer in Progress.

I programmed this game in with a React and TypeScript, and designed the page layout and UI/UX in collaboration with the team at TED.

Pixellence

A daily web game adaptation of nonograms puzzles made for TED Conferences. I put the finishing touches on a work in progress by Singfisher Games, and designed the web frontend and UI/UX.

The game is programmed in Godot, and the wrapper is written in React and TypeScript. I designed the page layout and UI/UX in collaboration with the team at TED.

Playable Ads

From 2019 to 2023, I worked in ad tech, designing and programming playable advertisements in Javascript. These ads took the form of bite-sized games and were designed to run in browsers on desktop and mobile.

Since these ads are no longer publicly available, I've compiled a video of my work here.

Highlandia

A mobile app built in Unity, designed to help teach English as a second language using gamified learning. Released on Google Play.

QluQuest

An infinite runner quiz game developed in Unity for mobile devices.

I worked with a small team at Create Neptune as a programmer and designer. I worked with an artist to revamp a rough prototype with new art, and added an infinite runner minigame.

Please Leave Me Alone, I Need to Poop
